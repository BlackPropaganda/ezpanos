# Command Tree

To ensure proper XML Command structure, it is neccessary to ensure each command class
* target
* schedule
* clear
* show
* set
Are properly parsed when considering the plaintext cli implementation of the plaintext command builder. This file represents the command tree structure for panOS base version 11 retrieved from [Palo Alto Networks](https://docs.paloaltonetworks.com/pan-os/11-0/pan-os-cli-quick-start/cli-command-hierarchy/pan-os-11-0-cli-ops-command-hierarchy).
